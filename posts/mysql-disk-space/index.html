计算1000万条数据占了多大内存

CREATE TABLE `table_name` (
  id bigint(20) not null auto_increment, -- 8字节
  detail varchar(2000), -- 
  createtime  datetime, -- 8 字节
  validity int default '0', -- 4字节
  primary key (id) -- 8字节
);

每个字段设计完成后占用的字节数

(8+8+8+4) * 1000 0000 = 2 8000 0000 字节 == 18G

电脑中存储常用的单位全称读法：

1Byte(Byte 字节) = 8Bit 1KB (Kilobyte 千字节) = 1024Byte， 1MB (Megabyte，兆字节，简称“兆”) = 1024KB， 1GB (Gigabyte，吉字节，又称“千兆”) = 1024MB， 1TB (Terabyte，太字节，或百万兆字节) = 1024GB，其中1024=2^10 ( 2 的10次方)。

电脑中，计数使用的都是二进制，它是以2的次方数来计算的。
