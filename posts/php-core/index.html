Php Core

2020-10-12

Zend API 随着时间不断向前进化，PHP 已变成为一种更健壮与普及的编程语言。随着 PHP 5 的推出，Zend Engine 2 (ZE2)也到来了。ZE2 带来了全新的面向对象编程(OOP)模型，以及大量 API 的改进。PHP 6 ，在此文档编写时还处于活跃的开发状态，引入了 Zend Engine 3 (ZE3)，将在语言中带来完全的 Unicode 支持。
