SQL HAVING Clause The SQL HAVING Clause The HAVING clause was added to SQL because the WHERE keyword could not be used with aggregate functions.

HAVING Syntax SELECT column_name(s) FROM table_name WHERE condition GROUP BY column_name(s) HAVING condition ORDER BY column_name(s); Demo Database Below is a selection from the “Customers” table in the Northwind sample database:

More HAVING Examples The following SQL statement lists the employees that have registered more than 10 orders:

Example SELECT Employees.LastName, COUNT(Orders.OrderID) AS NumberOfOrders FROM (Orders INNER JOIN Employees ON Orders.EmployeeID = Employees.EmployeeID) GROUP BY LastName HAVING COUNT(Orders.OrderID) > 10; The following SQL statement lists if the employees “Davolio” or “Fuller” have registered more than 25 orders:

Example SELECT Employees.LastName, COUNT(Orders.OrderID) AS NumberOfOrders FROM Orders INNER JOIN Employees ON Orders.EmployeeID = Employees.EmployeeID WHERE LastName = ‘Davolio’ OR LastName = ‘Fuller’ GROUP BY LastName HAVING COUNT(Orders.OrderID) > 25;
